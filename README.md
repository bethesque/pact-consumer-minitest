# Pact::Consumer::Minitest

Provides Pact Consumer support for Minitest.

## Installation

Add this line to your application's Gemfile:

    gem 'pact-consumer-minitest'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pact-consumer-minitest
